Lowongan Kerja Programmer PT. Arneva Teknologi Solusi

PT. Arneva Teknologi Solusi merupakan perusahaan yang bergerak dibidang Teknologi Informasi. Khususnya Pembuatan dan pengembangan Sistem Aplikasi /Apps.


Tanggung Jawab Pekerjaan :
Pembuatan, Pengembangan dan Maintenance Aplikasi Web (Frontend dan Backend)


Syarat Pengalaman :
Berpengalaman minimal 3 tahun didalam Pemrogramam Web


Keahlian :

  • – Menguasai Database adminstrasi, migrasi Mysql, Postgree
  • – Menguasai UI / UX Designer – Menguasai PHP, HTML, JavaScript Framework (JQuery/reactJS/AngularJS)
  • – Menguasai juga Framework CodeIgniter / Laravel
  • – Menguasai Pembuatan API, REST, JSON
  • – Menguasai manajemen kode (repository) seperti GIT
  • – Nilai tambah yang menguasai Installasi, Administrasi, troubleshoot Os Linux.


Kualifikasi :

  • – Pria / Wanita Usia Maksimal 35 tahun
  • – Pendidikan (semua jenjang atau jurusan)
  • – Bepengalaman didalam pembuatan dan pengembangan sistem aplikasi
  • – Berkepribadian baik, jujur, teliti dan cekatan
  • – Bisa bekerja sesuai dengan target / timeline
  • – Bisa bekerja mandiri ataupun didalam tim
  • – Memiliki integritas dan loyalitas yang tinggi

Waktu Bekerja : Jam 8.00 – 17.00
